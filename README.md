# StreamWord

Игра для стримеров и зрителей Twitch — угадай слово.

## features

- Генерация случайных слов из разных категорий
- Таймер и отсчёт времени
- Twitch-чат с сообщениями
- Определение победителя
- Звуковые эффекты при победе
- Конфетти
- Профиль пользователя
- Лидерборд
- Тёмная тема с неоновыми акцентами

## tech Stack

- Next.js 14
- React
- Zustand
- Framer Motion
- Tailwind CSS
- Space Grotesk + Rajdhani шрифты
- Font Awesome иконки
